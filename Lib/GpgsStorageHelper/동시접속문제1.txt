<Cloud저장,Local저장 동시 지원시 문제>

문제 : 다중 기기에서 `동시` 접속 시 게임 진행 상태에 따라 원하는 결과인 기기를 이용자가 선택해 저장할 수 있다. (갓챠, 무작위 요소에서 문제) 
	정책상 슬롯 저장방식이아닌 NewGame/Continue 만 지원하기때문에 모든 저장방식이 동기화 되어야한다.

1. 이문제를 해결하기 위해선 갓챠,무작위 요소가 진행된 이후엔 사용자가 원하든 원하지 않든 GameData 를 저장해야한다.
2. Local,Cloud 두개 클라우드를 동시 지원 할 경우 두 저장방식 모두 저장해야한다.
3. Local 저장의 경우 인터넷 연결이 않된 상태에서도 저장할 수 있어야 하지만 이경우 Cloud 저장도 시도해야해 문제가생긴다.

해결법
1.Local저장,Cloud저장 을 동시 지원하지않는다.

2. LocalSave와 CloudSave 를 동시지원한다.
	2.1. 갓챠 요소 시도시 Cloud 서버 연결을 확인한다 (Local 로만 즐기고 싶은 유저들은 즐길수 없다.)
	2.2. 갓챠 요소는 Cloud 서버에 연결하지 않으면 시도하지 못하게한다. (라이브러리 레벨에서 해결이 불가능해보인다.)

3.Cloud 저장을한번이라도 시도하면 Local저장을 불가능하게한다. (Local저장을 유저들이 이용하다 Cloud 이용을 할 것 같다.)



동호 의견 : 해결법 (1), 해결법 (3)
